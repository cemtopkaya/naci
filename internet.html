Internet & Network Access

Corporate or Guest Wired/Wireless Network

Network services (including intranet and internet) provided within the corporate offices are listed below
  1. FGNHQ Office Wireless Access - The wireless network FGNHQ in office supports most devices (802.11 a/b/g/n) and it is protected by NAC (Network Access Control) which is only available to employees.
  2. FEVZIGANDUR Guest Wireless Access - FEVZIGANDUR Guest is the wireless network that is provided to visitors in offices (HQ, Mersin, İzmir). If you have a visitor for a short period of time and you need a wireless connection, then this is the network you should use.
  3. Ethernet Access - Ethernet Network access is procted by NAC (Network Access Control) and it is only avaliable to employees.
  4. Remote Access/ VPN - VPN provides remote access to servers, data sources and applications.
  5. Internet Access powered by Vodafone - Internet usage is being monitored and filtered (URL/Content/Malware/Bandwith) to provide faster and secure service.
  6. MPLS Network for All Fevzigandur Offices powered by Vodafone - Mpls is a secure infrastructure service designed to provide continuous and reliabled data, voice and video communications with all offices across the country.
